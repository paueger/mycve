### description
A buffer overflow vulnerability was discovered in Tenda W20E V4.0br_V15.11.0.6. Attackers can exploit this vulnerability by specifying the value of `gotoUrl`. When `gotoUrl` is passed to `formPortalAuth` and copied into the fixed-size `redirect_url` buffer by `strcpy` without length validation, it may cause a buffer overflow.

### code
~~~c
  websDefineAction("PortalAuth", formPortalAuth);
        gotoUrl = websGetVar((Webs_1 *)wp, "gotoUrl", defaultGetValue__1);
        strcpy(redirect_url, gotoUrl);
~~~
