### description
A buffer overflow vulnerability was discovered in Tenda W20E V4.0br_V15.11.0.6. Attackers can exploit this vulnerability by specifying the value of `qosIndex`. When `qosIndex` is passed to `formQOSRuleDel` and copied into the fixed-size `input.src` buffer by `strcpy` without length validation, it may cause a buffer overflow.
